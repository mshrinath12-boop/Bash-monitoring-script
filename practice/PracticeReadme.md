# Practice Scripts

A collection of Bash practice scripts covering core concepts.

## Topics Covered

- Variables and constants
- Arrays
- Loops (for, while, until)
- Functions
- String operations
- File handling
- Case statements
- User input handling

## How to Run
chmod +x script_name.sh
./script_name.sh

These scripts were written while learning Bash scripting
